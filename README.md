# BACK-JAVA

Бекенд сервис на Java Spring Boot для проекта NECPGAME.

## Описание

Репозиторий `BACK-JAVA` содержит игровой сервер, обрабатывающий игровую логику, аутентификацию, базу данных и API для клиентских приложений.

## О проекте

**NECPGAME** - это MMORPG с элементами киберспортивного шутера, экстрактшутера и глубокой социальной структурой в стиле CYBERPUNK 2077.

### Технологии

- **Java 21** - современная версия языка
- **Spring Boot 3.2** - фреймворк для разработки
- **PostgreSQL 15** - база данных
- **Flyway** - управление миграциями БД
- **OpenAPI Generator** - генерация кода из OpenAPI спецификаций
- **Spring Data JPA** - работа с БД
- **Spring Security** - безопасность
- **JWT** - аутентификация

## Быстрый старт

### Требования

- Java 21+
- Maven 3.8+
- PostgreSQL 15+
- Docker (опционально, для БД)

### Установка

1. **Клонировать репозиторий:**
   ```bash
   git clone <repository-url>
   cd BACK-JAVA
   ```

2. **Запустить PostgreSQL:**
   ```bash
   docker-compose up -d
   ```

3. **Собрать проект:**
   ```bash
   mvn clean install
   ```

4. **Запустить приложение:**
   ```bash
   mvn spring-boot:run
   ```

### Проверка работы

Откройте в браузере: `http://localhost:8080/health`

Или через curl:
```bash
curl http://localhost:8080/health
```

API документация (Swagger UI): `http://localhost:8080/swagger-ui.html`

## API эндпоинты

## Разработка

### Генерация кода из OpenAPI

```bash
mvn clean generate-sources
```

Сгенерированный код будет в `target/generated-sources/openapi`

### Применение миграций

Миграции применяются автоматически при запуске приложения через Flyway.
